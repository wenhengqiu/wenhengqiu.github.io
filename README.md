# 数据闭环研究中心网站

## 关于

这是一个展示自动驾驶数据闭环研究项目的 GitHub Pages 网站。

## 访问地址

https://wenhengqiu.github.io/dataloop-research

## 本地开发

```bash
# 克隆仓库
git clone https://github.com/wenhengqiu/dataloop-research.git
cd dataloop-research

# 本地预览
python3 -m http.server 8000
# 或
npx serve .
```

## 技术栈

- HTML5
- CSS3 (原生)
- JavaScript (原生)
- 响应式设计
