## Info-Getter 全自动采集系统

### 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Info-Getter v1.0                         │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│  │ Fetcher  │→ │Translator│→ │ Publisher│→ │ Scheduler│   │
│  │  采集    │  │  翻译    │  │  发布    │  │  调度    │   │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘   │
│        ↓             ↓             ↓             ↓         │
│   RSS/API      OpenClaw/Kimi   GitHub Pages   每小时执行   │
│   12个源       自动翻译         自动部署       守护进程     │
└─────────────────────────────────────────────────────────────┘
```

### 采集流程

```
每小时自动执行：
1. 扫描 12 个信息源 (RSS/API)
2. 采集新文章 (标题、摘要、URL)
3. OpenClaw/Kimi API 翻译 (中文标题和摘要)
4. 质量评分 (0-1 分)
5. 分类打标 (ai|robotics|autonomous|zhuoyu)
6. 去重检测 (基于 URL)
7. 保存到 JSON 文件
8. 推送到 GitHub
9. GitHub Pages 自动部署
```

### 质量保障

- **去重机制**：基于 URL 自动去重
- **质量评分**：0-1 分，低于 0.6 不收录
- **分类审核**：AI 自动分类 + 人工校验
- **链接检测**：自动检测 404 无效链接
- **内容过滤**：自动过滤非 AI 相关内容

## 数据质量标准

### 收录标准

| 标准 | 要求 |
|------|------|
| **内容相关性** | 必须与 AI、机器人、自动驾驶、卓驭科技相关 |
| **来源可信度** | 官方博客、知名媒体、学术论文 |
| **时效性** | 优先收录 7 天内的新文章 |
| **质量评分** | 质量分 ≥ 0.6 才收录 |
| **去重** | 基于 URL 自动去重，不重复收录 |

### 排除内容

- ❌ 非 AI 相关的财经新闻（如滴滴财报、航空业动态）
- ❌ 虚假内容（如 GPT-5 未发布却宣称发布）
- ❌ 测试数据（如 example.com 链接）
- ❌ 低质量内容（如纯营销软文）

## 技术架构

### 前端
- **框架**：纯 HTML + CSS + JavaScript
- **样式**：CSS Grid + Flexbox 响应式布局
- **图标**：SVG + Emoji
- **存储**：localStorage（收藏、阅读历史）

## 每日信息总结功能

### 功能描述

每次采集新文章后，系统自动生成「今日要点」：

1. **提取核心事件**
   - 从今日文章中识别最重要的 3-5 条
   - 按质量分数和重要性排序

2. **生成要点摘要**
   - 一句话概括每条核心新闻
   - 提取关键数据和信息

3. **分类统计**
   - 今日新增文章总数
   - 各分类（AI/具身智能/自动驾驶/卓驭）占比

### 展示位置

- **首页顶部**：「🔥 今日要点」区域
- **状态页面**：「📅 今日更新」统计

### 数据格式

```json
{
  "date": "2026-03-14",
  "highlights": [
    "OpenAI 发布 GPT-5.4，性能提升 40%",
    "特斯拉 FSD V14 推送，城市导航改进",
    "卓驭科技完成新一轮融资，估值超 100 亿"
  ],
  "trend": "今日 AI 行业重点关注大模型性能提升和自动驾驶技术进展...",
  "stats": {
    "total": 12,
    "ai": 8,
    "autonomous": 2,
    "robotics": 1,
    "zhuoyu": 1
  }
}
```
